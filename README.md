#MiHorno3

MiHorno3 is an interactive mobile application designed to enhance the visitor experience at **Horno 3 Museo del Acero** in Monterrey, Mexico.

The application transforms the traditional museum visit into an **immersive digital experience** using **Augmented Reality (AR), gamification, and interactive challenges**.

## Project Overview

Visitors will be able to explore the museum while interacting with different industrial elements of the blast furnace through augmented reality missions.

The application allows users to:

* Scan museum structures
* Trigger AR-based experiences
* Complete engineering challenges
* Track progress and achievements
* Learn about steel production history
